## [0.1.0] - 2025-02-19

### Added

- Core allocation engine with Merton-style optimal risky share
- Human capital present value calculation with mortality adjustment
- Two-tier borrowing rate model for leverage support
- Four income models: flat, constant-growth, age-profile, CSV-based
- Strategy comparison against 60/40, 100-minus-age, and target-date funds
- Visualization suite: balance sheet waterfall, glide paths, sensitivity tornado, heatmaps
- CLI interface with YAML/JSON profile support
- Full report generation (JSON, Markdown, charts)
- Example profiles: young saver, mid-career, near-retirement
- Interactive tutorial notebook
